---
name: Badge text color rule
description: All badges must use white text regardless of variant/border color
type: design
---
Todas as badges (status, tags, chips de jogador, etc.) devem usar texto branco (`text-white`).
A diferenciação visual deve vir de border e/ou background sutil — nunca da cor do texto.
Não usar `text-green-400`, `text-amber-400`, `text-blue-400`, `text-gold`, etc. em badges.
